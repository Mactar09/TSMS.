Bienvenue dans TSMS(Tous Savoir sur Mon Site). TSMS est un site que j'ai conçu grâce à plusieurs langages. Dans ce site je vous détaillerai ces langages de programmations

HTML, ou HyperText Markup Language, est le langage standard utilisé pour créer des pages web. Il structure le contenu en utilisant des balises, qui définissent des éléments comme les titres, les paragraphes, les images et les liens. En somme, HTML est la base de toute création sur le web.

Le CSS, ou Cascading Style Sheets, est un langage de style utilisé pour décrire la présentation d'un document HTML ou XML. Il permet de contrôler l'apparence des éléments d'une page web, tels que les couleurs, les polices, les marges, les espacements et la disposition.

JavaScript est un langage de programmation dynamique et orienté objet, principalement utilisé pour le développement web. Il a été créé en 1995 par Brendan Eich et est devenu l'un des langages de programmation les plus populaires au monde.
